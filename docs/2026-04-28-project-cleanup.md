# Project Cleanup — File Audit & Dead Code Removal

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove session artifacts, compiled output, and dead source files from the ScopusResume repo, leaving a clean, minimal file tree with only files the app actually needs.

**Architecture:** Two phases — Phase 1 deletes obviously-safe files (session artifacts, build output, old plans) without any analysis. Phase 2 runs a grep-based import analysis on every ambiguous file before deletion; nothing gets removed unless its import count is confirmed zero.

**Tech Stack:** Bash (find, grep, rm), git
